# Next.js Template

Template Next.js modern dengan integrasi React 19, Auth.js, Tailwind CSS, Prisma, dan tools pendukung lainnya.

## Tech Stack

- React 19
- Next.js 15
- Auth.js v5
- Tailwind CSS 4
- Prisma ORM
- React Hook Form
- Zod v4
- ESLint
- Prettier

## Instalasi

1. Clone repository:

```bash
- git clone https://github.com/sdprdh/nextjs-template.git

- cd nextjs-template

- npm install
```
